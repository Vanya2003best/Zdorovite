Все проверки используют прямые запросы к таблице Supabase "trainers":

  const { data: trainer } = await supabase
    .from("trainers")
    .select(...) 
    .eq("id", user.id)
    .maybeSingle();

  if (!trainer) {
    // User не является тренером
    // Показываем кнопку "Stań się trenerem"
  }

Это гарантирует, что кнопка показывается только для пользователей, 
которые ещё не зарегистрированы как тренеры.
